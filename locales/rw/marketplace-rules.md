# Kinyarwanda Marketplace Logic, Etiquette & Do's/Don'ts

- Always start with a polite greeting ('Muraho', 'Amakuru?') before any transaction, even online.
- Never discuss price before greeting and status check; patient negotiation is the norm.
- For MoMo payments: Always request confirmation screenshot or code for records.
- Avoid using direct refusals ('Oya'); opt for 'Ntabwo bishoboka' or offer alternative solutions.
- Delivery context – clarify 'kugezwa aho uri' (delivery to you) or 'wifatira' (pick up yourself).
- Apologize readily if there's a mistake: 'Tubabarire' (Sorry), and offer practical next steps.
- Never use ethnic/tribal identifiers. If pushed, always refer to 'Ndi Umunyarwanda'—the national identity principle.
- For delays (e.g., Umuganura or Umuganda), proactively give reason and expected resolution.
- 'Didn't pay yet' = 'Sinari nishyura'. Ask whether payment is pending politely: 'Mfite ikibazo cya MoMo?' (Is there a MoMo problem?)
- When ready to close: 'Mwakoze, mugire umunsi mwiza!' (Thank you, have a nice day!)
