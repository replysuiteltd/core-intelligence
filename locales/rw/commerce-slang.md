# Kinyarwanda Business & Commerce Language Guide

| Ijambo/English         | Ibisobanuro (Meaning/Context)                                  |
|-----------------------|---------------------------------------------------------------|
| MoMo                  | MTN Mobile Money, main method of payment                       |
| Gupaya                | To pay instantly by mobile money                               |
| Code/Ikode            | Payment or confirmation code                                   |
| Ifoto/Screenshot      | Photo of payment confirmation ("Ohereza ifoto y'ubwishyu")     |
| Igiciro               | Price                                                          |
| Igiciro cya nyuma     | Final or negotiated price                                      |
| Kwishyura             | To pay (verb)                                                  |
| Umukiriya             | Customer                                                       |
| Uwagurishije          | Seller                                                         |
| Kugurisha             | To sell                                                        |
| Kwakira               | To receive (money/product)                                     |
| Gucuruza              | General term for trading (market/business context)             |
| Gucuruza online       | E-commerce (selling online or on social platforms)             |
| Kurangura             | To buy stock/wholesale                                         |
| Kwohereza kuri WhatsApp| Share/send via WhatsApp/Facebook                               |
| Inyemezabwishyu       | Invoice/Receipt                                                |
| Amasezerano y'akazi   | Business contract/employment offer                             |
| Akazi                 | Work, job (professional context: "ushaka akazi?")            |
| Ibikoresho by'ikoranabuhanga | Tech equipment (often, phone accessories, etc.)         |
| Kubitsa amafaranga    | Deposit money                                                  |
| Kuyishyura banki      | To pay via bank                                                |
| Serivisi              | Service (broad usage, e.g., "serivisi nziza")                |
| Gupakira              | To package (goods for delivery/shipping)                       |
| Gucyura               | To return (goods)                                              |

_Note: This is an evolving list—community/business partners can add new slang and current expressions._
