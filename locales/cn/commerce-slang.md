# 商务与支付用语 (Business & Payment Terms)

| 中文           | English        | 用法/Contexts                         |
| -------------- | -------------- | --------------------------------------|
| 转账           | Transfer       | 微信、支付宝、银行卡 (“扫码转账”)         |
| 微信支付        | WeChat Pay     | 日常主流支付方式                         |
| 支付宝         | Alipay         | 官方/线上购物                            |
| 二维码         | QR Code        | 扫码付款/Scan to pay                    |
| 下单           | Place order    | “可以帮我下单吗？”                       |
| 客户/买家      | Customer/buyer | “亲，欢迎光临!”                          |
| 优惠/折扣       | Promo/Discount | 活动期间专用                              |
| 订单编号       | Order number   | 付款/查询/售后                            |
| 评价/评论      | Review/Feedback| 客户留言                                  |
| 秒杀           | Flash Sale     | 限时抢购、折扣                            |
| 商家           | Seller         | 电商平台、微店                             |
| 售后           | After-sales    | 售后服务、退款相关                         |

_现代商务聊天以高效简洁为主，表情符号、点赞、“亲”字常用增进好感。_
