# Kirundi Business, Payments & Marketplace Language

| Kirundi           | French/English          | Context/Usage                          |
|-------------------|------------------------|----------------------------------------|
| EcoCash           | Paiement par mobile    | Main method, “Ohereza kuri EcoCash.”   |
| Kwishura          | To pay (payer)         | Transaction, “Warishuye?”              |
| Code y’ubwishyu   | Payment code           | “Numpa code w’umwisho wishyuye?”       |
| Ifoto ry’ubwishyu | Screenshot/payment pic | “Nohereza ifoto y’ubwishyu.”           |
| Igiciro           | Le prix/Price          | Ask: “Ni igiciro ingahe?”               |
| Kugurisha         | To sell                | General business/trade context         |
| Umuguzi           | Client                 | “Umuguzi ashimishijwe...”              |
| Umugurisha        | Seller                 | Shop owner or informal seller          |
| Kurungika         | To send                | Money/product delivery                 |
| Inyemezabwishyu   | Invoice/Receipts       | Business or government transaction doc |
| Gushumbusha       | To refund              | If mistake: “Gushumbusha amafaranga.”  |
| Gusubiza amafaranga| Return/refund funds    | Customer support resolution            |
