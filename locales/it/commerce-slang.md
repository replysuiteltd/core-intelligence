# Glossario Commercio e Pagamenti (IT)

| Italiano           | Utilizzo/Contesto                          |
|--------------------|--------------------------------------------|
| Bonifico           | Pagamento tramite IBAN                     |
| Pagamento alla consegna | Contrassegno, pay on delivery      |
| Sconto             | Richiesta “Mi fa uno sconto?” (“Can I get a discount?”)
| Offerta lampo      | Flash sale, promo di breve durata          |
| Cliente abituale   | Cliente ricorrente, trattare con calore    |
| Magazzino/Disponibilità | Stock/inventory check               |
| Spedizione         | Shipping/delivery ("Come spedite?")       |
| Fattura            | Invoice/official receipt                   |
| Codice sconto      | Coupon/discout code                        |
| Ricevere/Restituire| Receive/refund goods                      |

__Nota:__ WhatsApp e app mobile usatissime per conferma spedizione/pagamento in Italia 2026.
