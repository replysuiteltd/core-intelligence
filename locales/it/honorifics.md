# Titoli Commerciali & Politeness (IT)

- Signore/Signora: Adult/elder, very formal.
- Ragazzo/a: Peer/youth, informal (“ciao ragazza!”)
- Dottore/Dottoressa: For professionals, high respect.
- Caro/a + first name: Loyal customer, repeat rapport.
- Capo/Boss: Urban, for friendly regulars (not always literal boss).

__Usage__:
Always default to formal “Lei” unless user is clearly informal; Italian business culture values initial distance.
